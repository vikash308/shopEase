# 🛍️ ShopEase – Dynamic E-Commerce Website

Welcome to **ShopEase**, a dynamic and responsive e-commerce platform built with **HTML**, **CSS**, and **JavaScript**, featuring live product search powered by the **DummyJSON API**. ShopEase allows users to explore products, manage their cart, wishlist, and orders — all in a beautifully designed interface.

## 📦 Features

- 🔍 **Live Product Search** using [DummyJSON API](https://dummyjson.com)
- 🛒 **Shopping Cart**, **Wishlist**, and **Order Management** using `localStorage`
- 📂 **Category-based Product Browsing**
- 📦 **Order form** with delivery details popup
- 📄 **Dynamic Product Pages**
- 🌗 **Responsive design** for mobile & desktop
- 🔁 **Reusable Navbar Component**
- 🔐 **Login Page** ready for backend integration
